## Requirement to install Game Launcher

- [X] .NET Framework 4.5
- [X] Windows Vista SP2 or Newer
- [X] Windows Firewall Exception
- [x] Custom Directory (No Programs Files or x86 anymore)
- [ ] Admin Rights
